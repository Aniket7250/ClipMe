## Installation

1. Download the DMG file above
2. Open the downloaded `ClipMe-1.0.0.dmg` file
3. Drag ClipMe to your Applications folder
4. Launch ClipMe from Applications
5. Grant necessary permissions when prompted

## System Requirements

- macOS 13.0 (Ventura) or later
- Apple Silicon (M1/M2/M3) or Intel processor

## First Launch

After launching ClipMe for the first time:
1. You may need to right-click and select "Open" if macOS shows a security warning
2. Grant Accessibility permissions in System Settings â†’ Privacy & Security
3. The ClipMe icon will appear in your menu bar

## Quick Start

- **Global Hotkey**: Press `Cmd + Shift + V` to open the clipboard history
- **Pin Items**: Click the pin icon to keep important items
- **Clear History**: Use "Clear All" to remove unpinned items
